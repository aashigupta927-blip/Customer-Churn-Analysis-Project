# 📊 Customer Churn Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing customer churn behavior using Python,SQL & Power Bi. The goal is to identify patterns and key factors that contribute to customer attrition.

## 📂 Dataset

The dataset contains customer details such as:

* Gender
* Subscription Plan
* Tenure
* Payment Method
* Internet Service
* Satisfaction Score
* Monthly & Total Charges

## 📊 Dashboard Features

* KPI Cards (Total Customers, Churn Customers, Churn Rate)
* Churn analysis by Satisfaction Score
* Churn by Payment Method
* Churn by Internet Service
* Tenure-based churn distribution
* City-wise revenue analysis

## 🔍 Key Insights

* Customers with low satisfaction scores are more likely to churn
* Electronic check users show the highest churn rate
* Fiber optic users have higher churn compared to DSL users
* New and mid-tenure customers are at higher risk

## 🛠 Tools & Technologies

* Power BI
* DAX
* Data Cleaning & Transformation
